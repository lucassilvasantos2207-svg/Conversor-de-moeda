# Conversor de Moedas em Java

Projeto desenvolvido em Java com menu interativo para conversão de moedas.

## Funcionalidades

- Conversão Real ↔ Dólar
- Conversão Real ↔ Euro
- Conversão Real ↔ Libra
- Conversão Real ↔ Peso Argentino
- Conversão avançada com mais de 20 moedas

## Tecnologias

- Java
- Scanner
- Estruturas de decisão (switch)
